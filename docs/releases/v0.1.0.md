# v0.1.0 - Windows 安装包首版

这是视频应用进程管理工具的第一个 Windows 安装包版本。用户无需安装 Rust、Cargo 或开发环境，下载并运行安装包即可使用。

## 下载文件

- `VideoProcessManagerSetup-0.1.0.exe`

## 安装包信息

- 版本：`0.1.0`
- 平台：Windows
- 安装包大小：约 3.56 MB
- SHA256：`3F8CFC5395A03AD09397D8364FC3440FFC69ED879973DD55EBEFED04FA778F5A`
- 默认安装目录：`%LOCALAPPDATA%\VideoProcessManager`
- 当前未做代码签名，Windows 可能提示“未知发布者”。

## 主要功能

- 搜索并关闭预设视频/直播类应用进程。
- 内置预设：腾讯视频、爱奇艺、芒果TV、优酷、哔哩哔哩、抖音。
- 支持自定义应用和多个进程名。
- 搜索结果按应用分组，可展开查看子进程。
- 任务管理器式列表展示：名称、类型、PID、CPU、内存、状态。
- 支持单选、应用分组批量选择、全选、反选和一键关闭。
- Windows 下普通关闭失败时，会在校验 PID + 进程名后使用 `taskkill /PID /T /F` 兜底。

## 配置文件

安装目录包含：

```text
video-process-manager.exe
presets.toml
config.toml
```

- `presets.toml`：预设应用进程名配置。
- `config.toml`：用户自定义应用配置。
- 旧版本保存在用户配置目录的 `config.toml` 仍可被兼容读取。
